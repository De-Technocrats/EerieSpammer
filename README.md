1. Desktop or Laptop. (Android not supported.)
2. 500 MB internet.
3. Basic computer knowledge requried.

>### [+]COMMANDS</b></i>
>>#### 1. For Linux users.
```

```
>>#### 2. For Windows users.

Install git from  [here](https://gitforwindows.org/)

Now run following commands.
```

```
